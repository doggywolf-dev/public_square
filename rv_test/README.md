# Отчет о выполнении тестового задания

## Обучение модели
На данных открытого датасета COCO с добавлением собственных, специально размеченных данных, была обучена модель для детекции трех классов: airplanes, birds и kites.

Подробная информация о характеристиках модели, метриках, ходе обучения, исходных данных и использованных для их подготовки скриптах приведена [в разделе с моделью](https://github.com/doggywolf-dev/public_square/blob/main/rv_test/models/README.md)

##
Для отображения результатов работы модели на кадрах видео было разработано приложение bin_pkrv_test на C++.  

Сборка и работа приложения выполняется в среде Linux. Для корректной сборки и работы необходимо наличие следующих установленных компонент:
	CMake (версия ≥ 3.22)
	Компилятор C++ с поддержкой C++17 (например, GCC 9+, Clang 10+)
	OpenCV (с заголовочными файлами и библиотеками)
	ONNX Runtime (Linux‑версия для x64, версия 1.23.2)
	pthread (обычно входит в систему)

Для сборки проекта необходимо:

1.В корневой директории проекта (где лежит основной CMakeLists.txt) создать директорию для сборки:

mkdir build && cd build

Далее выполнить 

cmake ..
make

После сборки в директории build/ появится:

исполняемый файл: bin_pkrv_test;

Запуск:

./bin_pkrv_test --video <путь к видео> --model <путь к модели> -w <число кадров для усреднения, можно пропустить> -k <размер окна для фильтра скользящей медианы, тоже можно пропустить>
